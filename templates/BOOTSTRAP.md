# BOOTSTRAP.md - Инструкция после пробуждения

> Этот файл грузится автоматически. Выполняй СРАЗУ после старта/компактификации.

## 1. Прочитай handoff
```
read memory/handoff.md
```
Там: тема, решения, TODO, файлы, контекст. Это твой "save game".

## 2. Прочитай сегодняшний дневник
```
read memory/YYYY-MM-DD.md  (подставь текущую дату)
```
Автодневник пишется 2x/день. Там контекст дня.

## 3. Верификация
После чтения handoff - сверь с последними сообщениями:
```
sessions_history(sessionKey="agent:main:main", limit=10, includeTools=false)
```
Проверь: совпадает ли тема handoff с реальным последним разговором?
- Если да - продолжай работу
- Если нет (handoff устарел) - ориентируйся на sessions_history

## 4. Если handoff пустой или старый
```
sessions_history(sessionKey="agent:main:main", limit=20, includeTools=false)
```
Посмотри последние сообщения - восстанови контекст.

## 5. Продолжай работу
Если в handoff есть незавершённая задача - продолжи.
Если владелец ждёт ответ - ответь.
Если ничего не висит - МОЛЧИ (NO_REPLY), не пиши "я проснулся".
