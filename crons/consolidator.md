# Consolidator - Еженедельная консолидация памяти

> Запускать раз в неделю: воскресенье 03:00
> Модель: лёгкая (Sonnet / GPT-4o-mini)

## Промпт для крона

```
Еженедельная консолидация памяти.

1. Прочитай все daily notes за последние 7 дней:
   ls memory/20??-*.md - прочитай каждый

2. Извлеки паттерны:
   - Повторяющиеся темы/задачи
   - Решения которые стоит зафиксировать
   - Новые факты о владельце/проектах

3. Обнови файлы:
   - memory/decisions/ - новые решения (если были)
   - memory/core/ - новые постоянные факты (если были)
   - MEMORY.md - обнови если изменились приоритеты или ключевые факты

4. Проверь MEMORY.md на размер:
   Если > 3000 символов - сожми, убери устаревшее

Правила:
- НЕ перезаписывай существующие файлы в core/ и decisions/ - ДОПОЛНЯЙ
- Проверяй перед записью: нет ли уже этого факта
- Если за неделю ничего важного - NO_REPLY
```
